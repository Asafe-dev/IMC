# IMC
Calculadora de IMC em Python
Esse projeto foi desenvolvido por mim para praticar Python e fortalecer minha lógica de programação.
O programa solicita o peso e a altura do usuário, calcula automaticamente o Índice de Massa Corporal (IMC) e informa a classificação do resultado.

Funcionalidades
•Entrada de peso
•Entrada de altura
•Cálculo do IMC
•Classificação do resultado (abaixo do peso, normal, sobrepeso e obesidade)

Objetivo
Treinar conceitos fundamentais como entrada de dados, operações matemáticas e estruturas condicionais.

Este é um dos meus primeiros projetos em Python e faz parte da minha evolução como desenvolvedor, primeiro de muitos.
